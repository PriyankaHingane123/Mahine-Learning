 ###  What is Machine Learning?
 - Machine Leatning (ML) is a subset of Artificial Intelligence that enables machines to learn from data and make predictions.

 ###  Types of Machine Learning:
 1. Supervised Learning
 - Supervised learning is a type of machine learning where the model is trained using labeled data (input+ output)

 2. Unsupervised Learning
 - Unsupervised learning is used when the data is unlabeled (no output column).

 ### Regression Models:
   - what is Regression?
   --> Regression is a supervised learning technique used to predict continuous numerical values.

 ### Linear Regression Model
  - Linear Regression is a regression algorithm that shows a linear relationship between input and output variables.
  - Mathematical Equation: y= mX + c
  - where:
    y= dependent variable
    x= independent variable
    m= slope
    c= intercept

1. Simple Linear Regression
- Simple Linear Regression uses one independent variable to predict a dependent variable.

2. Multiple Linear Regression
- Multiple Linear Regression uses more than one independent variable to predict a dependent variable.
- Equation: y= m1X1 + m2X2 + m3X3 + c
